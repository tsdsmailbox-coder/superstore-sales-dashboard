# Superstore Sales Dashboard

**Personal Project | Python · Excel**

## Overview
Analyzed 4 years of US superstore sales data (2011–2014) to identify
revenue trends across product categories and time periods.


## What I Did
- Loaded and cleaned raw sales data using Python (pandas)
- Analyzed revenue by product category and year
- Built an interactive Excel dashboard with KPIs, bar chart, and line chart


## Key Findings
- **Total Revenue (2011–2014):** $12.6M
- **Top Performing Category:** Technology
- **Peak Sales Year:** 2014


## Tools Used
Python (pandas, matplotlib) · Excel (PivotTables, Charts, Dashboard)
