Superstore Sales Dashboard
Overview: Analysis of 4 years of US superstore sales data (2011 - 2014) to identify revenue trends across product categories and time periods.
What I did
•	Loaded and cleaned raw sales data using Python and pandas
•	Analysed revenue by product category and year
•	Built an interactive dashboard in Excel with KPIs, a bar chart, and a line chart
Key Findings
•	Total revenue across 4 years: $12.6M
•	Top performing category: Technology
•	Peak sales year: 2014
Tools Used Python (pandas, matplotlib) · Excel (PivotTables, Charts, Dashboard)
